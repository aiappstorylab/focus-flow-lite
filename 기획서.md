## 1. 프로젝트 개요

- **서비스명:** 포커스 플로우 Lite (Focus Flow Lite)
- **개발 목표:** 서버 없이 동작하는 독립형 학습 타이머 웹 앱 구축
- **핵심 컨셉:** "설치가 필요 없는, 나만의 몰입 대시보드"

---

## 2. 주요 기능 (Frontend Only)

### ① 하이브리드 타이머

- **뽀모도로 & 커스텀:** 25/5분 법칙 외에도 사용자가 자유롭게 집중/휴식 시간을 설정.
- **백그라운드 유지:** 브라우저 탭이 비활성화되어도 `Web Worker`를 이용해 끊김 없는 시간 추적.
- **시각적 피드백:** 시간이 흐름에 따라 배경색이 변하거나, 진행률을 나타내는 프로그레스 바(Circle Progress) 표시.

### ② 로컬 데이터 엔진 (No-DB)

- **Persistence:** `localStorage`를 활용해 새로고침을 해도 설정값과 오늘 공부한 시간이 유지됨.
- **Daily Log:** 날짜별 학습 시간을 배열 형태로 저장하여 최근 7일간의 기록 관리.
- **Data Export:** 사용자가 원할 경우 전체 학습 데이터를 `JSON` 파일로 내보내거나 가져오기(Backup/Restore).

### ③ 몰입 지원 기능

- **White Noise:** `Audio HTML5 API`를 사용하여 빗소리, 모닥불 소리 등 로컬 오디오 재생.
- **브라우저 알림:** 집중 시간이 종료되면 시스템 알림(Notification) 송출.
- **Zen 모드:** 타이머 외의 모든 UI 요소를 숨기고 현재 시간만 크게 표시.

## 3. 기술 스택 (Simple & Modern)

| **구분** | **기술** | **이유** |
| --- | --- | --- |
| **Framework** | **React (Vite)** | 컴포넌트 기반 개발 및 빠른 빌드 속도 |
| **State** | **Zustand** | Redux보다 가볍고, 타이머 로직을 직관적으로 관리 |
| **Styling** | **Tailwind CSS** | 별도의 CSS 파일 없이 유틸리티 클래스로 빠른 UI 구현 |
| **Charts** | **Recharts** | 공부 기록을 시각화하기 가장 쉬운 React 라이브러리 |
| **Storage** | **LocalStorage** | 서버 없이 브라우저에 데이터를 반영구적으로 저장 |

---

## 4. UI/UX 화면 설계

1. **메인 대시보드:** 중앙에 큰 원형 타이머, 하단에 시작/일시정지/초기화 버튼 배치.
2. **설정 패널:** 집중 시간, 휴식 시간, 알림음 선택, 다크모드 전환.
3. **통계 탭:** 최근 일주일간의 학습량을 막대그래프로 시각화.
4. **플레이리스트:** 몰입을 돕는 화이트 노이즈 선택 슬라이더.

## 5. 데이터 구조 (Schema)

`localStorage`에 저장될 데이터 구조 예시입니다.

```json
{
  "focus_settings": {
    "work": 1500,  // 25분 (초 단위)
    "break": 300,  // 5분
    "theme": "dark"
  },
  "focus_history": [
    { "date": "2026-02-10", "minutes": 120 },
    { "date": "2026-02-11", "minutes": 45 }
  ]
}
```

## 6. 개발 단계 (Roadmap)

- **1단계:** React + Zustand를 활용한 기본 카운트다운 로직 구현.
- **2단계:** Tailwind CSS를 활용한 타이머 및 설정 UI 디자인.
- **3단계:** LocalStorage 연동 및 Recharts를 이용한 통계 화면 추가.
- **4단계:** 오디오 알림 및 Web Worker를 이용한 성능 최적화.

## 7. 제약 사항

- 이미지,아이콘,폰트, 사운드 파일의 경우 라이센스가 무료이면서 상업적으로 사용가능해야하며, 저작권 고지를 할 필요가 없는 리소스만 사용할 것.
